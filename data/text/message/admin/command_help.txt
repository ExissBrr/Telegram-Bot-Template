Справочник пользователя.

Тебе доступны команды:
/help: Получить справочник.

/start: "Получить главную клавиатуру.

/settings: Настройки аккаунта

/bun [user_id] [report]: Блокировка пользователя по его id.
[user_id] - id пользователя
[report] - причина, которую увидет пользователя, по умолчанию "Не указано"

/unbun [user_id]: Разблокировка пользователя по его id.
[user_id] - id пользователя
